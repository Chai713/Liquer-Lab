# Liqueur Lab Pro

A dark, offline-first recipe and formulation workstation for craft liqueurs.

## Run locally

1. Install [Node.js 20+](https://nodejs.org/).
2. In this folder run `npm install`.
3. Run `npm run dev` and open the shown local address.

## Included

- Dashboard with ABV and formulation charts
- ABV, dilution, sugar, coffee and limoncello calculators
- Local recipe builder, presets, favorites, duplicate, edit, delete and search
- Inventory tracking and low-stock alerts
- Printable PDF label generator
- Persistent browser storage for recipes, inventory and preferences

All data is stored only in the browser's local storage. Use the browser's normal site-data controls to clear it.
